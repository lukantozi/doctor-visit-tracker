# doctor-visit-tracker

Django web app for doctors to track their patients and visit history.

- Secure login for each doctor (Django auth)
- Patient records with contact details and date of birth
- Visit records linked to patients and doctors, with date and notes
- Search and filter across patients and visits
- Per-user data isolation: doctors only see and edit their own patients and visits
- Basic success messages and custom error pages

## Tech

Django, SQLite, Django class-based views, Django forms, Bootstrap (templates)
